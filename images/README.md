# 图片文件夹说明

## 📷 需要添加的图片

### 1. 个人照片
- **文件名**: `profile.jpg` 或 `profile.png`
- **建议尺寸**: 400×400 像素（正方形）
- **说明**: 你的个人照片，会以圆形显示

### 2. 论文缩略图
- **位置**: `papers/` 子文件夹
- **建议尺寸**: 400×300 像素（4:3 比例）
- **命名**: `paper1.png`, `paper2.png` 等
- **说明**: 论文的示意图、架构图或结果可视化

## 📐 图片规格建议

| 类型 | 尺寸 | 格式 | 大小建议 |
|------|------|------|----------|
| 个人照片 | 400×400px | JPG/PNG | < 200KB |
| 论文缩略图 | 400×300px | PNG | < 150KB |

## 🎨 图片优化建议

1. **压缩图片**: 使用 [TinyPNG](https://tinypng.com/) 或 [Squoosh](https://squoosh.app/) 压缩图片
2. **适当裁剪**: 确保重要内容在图片中央
3. **统一风格**: 保持所有论文缩略图风格一致

## 📝 如何添加图片

1. 将个人照片重命名为 `profile.jpg` 并放入当前文件夹
2. 在 `papers/` 文件夹中添加论文缩略图
3. 在 `index.html` 中引用图片路径

## ⚠️ 注意事项

- 如果暂时没有图片，代码中已内置占位符功能
- 图片加载失败会自动显示 emoji 占位符
- 建议使用高质量图片以获得最佳显示效果
