# 📖 快速使用指南

## 🎯 第一步：修改基本信息（5分钟）

打开 `index.html`，按照下面的标记修改：

### 1. 修改标题和个人信息
```html
<!-- 第7行左右 -->
<title>张三 - 个人主页</title>  <!-- 改成你的名字 -->

<!-- 第65行左右 -->
<h1>张三</h1>                    <!-- 改成你的名字 -->
<p class="title">博士研究生</p>  <!-- 改成你的职位 -->
<p class="affiliation">某某大学...</p>  <!-- 改成你的单位 -->
```

### 2. 修改社交链接
```html
<!-- 第72行左右 -->
<a href="mailto:your.email@example.com">📧 Email</a>
<a href="https://github.com/yourusername">💻 GitHub</a>
<!-- 把链接改成你自己的 -->
```

### 3. 填写研究兴趣
```html
<!-- 第95行左右 -->
<ul>
    <li>人工智能与机器学习</li>  <!-- 改成你的研究方向 -->
    <li>深度学习理论与应用</li>
    <!-- 添加或删除条目 -->
</ul>
```

### 4. 写个人简介
```html
<!-- 第107行左右 -->
<p>
    我是某某大学...  <!-- 改成你的简介 -->
</p>
```

## 🎯 第二步：添加论文（10分钟）

### 复制这个模板，修改内容：

```html
<div class="paper-item">
    <img src="images/papers/你的论文图.png" alt="论文1" class="paper-thumbnail">

    <div class="paper-info">
        <h4 class="paper-title">你的论文标题</h4>

        <p class="paper-authors">
            <strong>你的名字</strong>, 合作者1, 合作者2
        </p>

        <p class="paper-venue">
            <em>会议或期刊名称</em>, 2024
        </p>

        <p class="show_paper_citations" data="paper_id_1">
            引用数: 加载中...
        </p>

        <div class="paper-links">
            <a href="论文PDF链接" target="_blank">📄 PDF</a>
            <a href="代码链接" target="_blank">💻 Code</a>
            <!-- 添加或删除链接 -->
        </div>
    </div>
</div>
```

### 需要修改的地方：
1. 图片路径（如果有的话）
2. 论文标题
3. 作者列表（自己的名字用 `<strong>` 加粗）
4. 发表信息
5. PDF、代码等链接

## 🎯 第三步：添加教育经历、奖项等（5分钟）

### 教育经历模板：
```html
<div class="education-item">
    <div class="edu-timeline">
        <span class="edu-year">2021 - 至今</span>  <!-- 时间 -->
    </div>
    <div class="edu-content">
        <h4>博士研究生 - 计算机科学与技术</h4>  <!-- 学位专业 -->
        <p class="edu-school">某某大学</p>  <!-- 学校 -->
        <p class="edu-advisor">导师: XXX 教授</p>  <!-- 导师 -->
        <p class="edu-desc">研究方向: ...</p>  <!-- 简介 -->
    </div>
</div>
```

### 荣誉奖项模板：
```html
<div class="honor-item">
    <span class="honor-year">2024</span>  <!-- 年份 -->
    <div class="honor-content">
        <h4>国家奖学金</h4>  <!-- 奖项名称 -->
        <p>教育部</p>  <!-- 颁发单位 -->
    </div>
</div>
```

## 🎯 第四步：添加图片（可选）

1. **个人照片**：
   - 放到 `images/` 文件夹
   - 命名为 `profile.jpg`

2. **论文图片**：
   - 放到 `images/papers/` 文件夹
   - 如 `paper1.png`, `paper2.png`

💡 **提示**：如果暂时没有图片，会自动显示占位符，不影响使用！

## 🎯 第五步：本地预览

### 方法1：直接打开
双击 `index.html` 用浏览器打开

### 方法2：本地服务器（推荐）
```bash
# Python 3
python -m http.server 8000

# 然后浏览器访问 http://localhost:8000
```

## 🎯 第六步：部署上线（可选）

### 部署到 GitHub Pages：

1. 在 GitHub 创建仓库 `你的用户名.github.io`
2. 上传所有文件到仓库
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://你的用户名.github.io`

详细步骤见 `README.md`

## 🆘 遇到问题？

### 常见问题：

**Q: 样式显示不正常？**
A: 检查文件结构，确保 `css/style.css` 和 `js/main.js` 在正确位置

**Q: 图片不显示？**
A: 检查图片路径，或暂时忽略（会显示占位符）

**Q: 想改颜色？**
A: 在 `css/style.css` 中搜索 `#0366d6`（蓝色）替换为你喜欢的颜色

**Q: 想删除某个区块？**
A: 在 `index.html` 中注释掉或删除对应的 `<section>` 标签
