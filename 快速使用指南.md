# 📖 快速使用指南

本指南帮助您在30分钟内完成个人学术主页的基础配置。

---

## 🎯 第一步：环境配置（5分钟）

### 1. 安装Pytho

```bash
pip install -r scripts/requirements.txt
```

### 2. 配置API密钥（可选）

```bash
# 复制配置模板
cp .env.example .env

# 编辑 .env 文件，填入API密钥
# OPENAI_API_KEY=sk-...
```

**说明**：
- API仅用于自动提取论文元数据
- 不配置也可以，手动编辑JSON文件即可
- PDF封面提取和头像准备无需API

---

## 🎯 第三步：添加论文（10分钟）

### 方法1：使用批量处理（推荐）

如果有论文PDF：

```bash
# 1. 将PDF放入文件夹
cp your_paper.pdf images/raw-papers/

# 2. 运行批量处理
python scripts/batch_processor.py

# 自动完成：
# ✅ 提取PDF完整首页为封面
# ✅ AI提取元数据（标题、作者、期刊等）
# ✅ 添加到 data/publications.json
# ✅ 生成News条目
```

### 方法2：手动编辑JSON

编辑 `data/publications.json`：

```json
{
  "publications": [
    {
      "id": "your_paper_2024",                    // 唯一ID
      "title": "你的论文标题",
      "authors": ["张三", "李四*"],                // *表示通讯作者
      "author_note": "*corresponding",
      "venue": "期刊或会议名称",
      "year": 2024,
      "volume": "12",
      "pages": "123-456",
      "type": "journal",                          // journal或conference
      "status": "published",                      // published或under_review
      "badges": [],                               // 如 ["🏆 Best Paper"]
      "image": "images/papers/your_paper.png",    // 封面图片路径
      "links": {
        "pdf": "https://...",                     // PDF链接
        "doi": "https://doi.org/...",             // DOI链接
        "code": "https://github.com/..."          // 代码链接（可选）
      },
      "citation_key": "your_paper_2024"           // Google Scholar引用key
    }
  ]
}
```

## 🎯 第四步：准备头像（5分钟）

### 使用图片编辑工具

推荐工具：
- **Photoshop** / **GIMP** - 专业编辑
- **在线工具** - [remove.bg](https://www.remove.bg/)
- **系统工具** - Windows画图 / macOS预览

### 步骤：
1. 打开您的照片
2. 裁剪为正方形（1:1比例）
3. 调整尺寸为400×400像素
4. 保存为 `images/profile.jpg`

💡 **提示**：暂时没有头像也可以，网站会显示占位符。

## 🆘 常见问题

### Q: 必须配置API密钥吗？

**A**: 不是必须的。

- ✅ PDF封面提取：无需API，自动提取完整首页
- ✅ 头像准备：无需API，手动使用图片工具
- ⚠️ 元数据提取：需要API，或手动编辑JSON

### Q: 如何批量添加多篇论文？

**A**: 使用批量处理脚本。

```bash
# 将所有PDF放入文件夹
cp paper1.pdf paper2.pdf paper3.pdf images/raw-papers/

# 运行批量处理
python scripts/batch_processor.py

# 自动处理所有PDF
```

### Q: 图片不显示怎么办？

**A**: 检查以下几点：

1. 图片路径是否正确（相对于index.html）
2. 图片文件是否存在
3. 文件名是否匹配JSON中的配置
4. 使用本地服务器预览（不要直接双击HTML）

### Q: News内容如何生成？

**A**: 自动生成。

- 添加论文 → 自动生成发表消息
- 添加数据集 → 自动生成发布消息
- 添加奖项 → 自动生成获奖消息

手动编辑 `data/news.json` 也可以。

### Q: 如何修改网站配色？

**A**: 编辑 `css/style.css`。

```css
/* 主色调 */
#0366d6  /* 蓝色 - 链接、按钮 */
#28a745  /* 绿色 - 数据集 */
#6f42c1  /* 紫色 - 代码工具 */
```

全局替换颜色值即可。

## 🎨 进阶功能

### 1. 内容格式化助手

使用AI辅助格式化内容：

```bash
# 交互式添加论文
python scripts/content_formatter.py --type publication

# 交互式添加数据集
python scripts/content_formatter.py --type dataset

# 交互式添加奖项
python scripts/content_formatter.py --type award
```

### 2. Google Scholar引用统计

更新 `google-scholar-stats/gs_data.json`：

```json
{
  "citedby": 1250,
  "publications": {
    "your_paper_2024": {
      "num_citations": 145,
      "title": "你的论文标题"
    }
  }
}
```

页面会自动显示引用数。

### 3. 自定义域名

创建 `CNAME` 文件：

```bash
echo "yourdomain.com" > CNAME
git add CNAME
git commit -m "Add custom domain"
git push
```

在域名服务商配置DNS：
- A记录指向：185.199.108.153
- 或CNAME指向：yourusername.github.io
